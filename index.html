TrajCakeWars

Moteur de mini-jeu Cake Wars 1v1 complet, prêt pour la production. Conçu pour la performance, la personnalisation et l'intégration avec n'importe quel serveur.

🏟️

Multi-Arènes

Parties simultanées illimitées. Chaque match tourne dans un monde temporaire cloné.

⚙️

100% Configurable

Items des shops, récompenses, messages, générateurs — tout dans config.yml.

🔌

Compatible tout plugin

Citizens, FancyNpcs, ZNPCsPlus, Vault, LuckPerms, TAB, ou un simple villageois renommé.

🔒

Sessions privées

Partie privée avec code 6 caractères. Défie tes amis.

💀

Pas de death screen

Système de mort custom. Jamais d'écran "Ressusciter". Respawn fluide avec countdown.

🧱

Map protégée

TNT ne détruit que les blocs posés. Le terrain original est intouchable.

Démarrage rapide

1. Préparer le template

# Copie manuelle ou en jeu :
/cw savetemplate desert

2. Configurer l'arène

/cw create desert
/cw tp desert                   # TP dans le template
/cw set desert spawn1
/cw set desert spawn2
/cw set desert cake1
/cw set desert cake2
/cw set desert gen1
/cw set desert gen2
/cw set desert diamond           # Répéter pour chaque gen
/cw set desert lobby
/cw setname desert §6§lDésert
/cw settemplate desert desert
/cw back
/cw setlobby                     # Au point de retour lobby
/cw enable desert
/cw save

Créer un template

Un template = dossier monde dans cw_templates/. Chaque partie le clone. L'original reste intact.

Méthode 1 : Copie manuelle

serveur/cw_templates/desert/   ← dossier monde complet

Méthode 2 : En jeu

/cw savetemplate desert
Important : Le template doit contenir les PNJ shops (entités clonées avec le terrain).

Configurer une arène

/cw tp <arène> pour te TP dans le template. /cw back pour revenir.

Paramètres

/cw setname <arène> <nom>          # Nom affiché (supporte §)
/cw seticon <arène> <MATERIAL>     # Icône GUI
/cw settemplate <arène> <nom>      # Lier au template
/cw set <arène> diamond clear     # Reset gens diamant

Déroulement

/cw → Menu GUI → Jouer / Privée
→ 2 joueurs matchés → Clone template → TP
→ Freeze 5s → GO ! → Générateurs actifs
→ PvP + Manger cake adverse (7 parts, 1s cd)
→ Cake détruit → prochaine mort = éliminé
→ Vainqueur → Récompenses → TP lobby → Monde supprimé

Système de mort

  • Dégâts fatals interceptés et annulés (pas de death screen)
  • Joueur passe en Spectateur avec countdown titre
  • Respawn avec épée de base après le countdown
  • Si cake détruit → élimination définitive
  • Guard anti-double : ignoré si déjà en spectateur

TNT & Explosions

Seuls les blocs posés par les joueurs sont détruits. Le terrain template est protégé.

Config — Général

settings:
  prefix: "&d&lCakeWars &r&7» &r"
  lobby-world: "world"
  lobby-location:     # Défini par /cw setlobby
    world/x/y/z/yaw/pitch

game:
  countdown: 5         # Freeze avant partie
  respawn-time: 5      # Countdown respawn
  max-duration: 600    # Max secondes (0=infini)

Config — Générateurs

generators:
  iron:
    interval: 40   # Ticks (20=1s)
    max: 32
  diamond:
    interval: 600
    max: 4

Config — Récompenses

rewards:
  enabled: true
  winner:
    commands:
      - "give %winner% diamond 5"
      - "eco give %winner% 100"
    message: "&6⭐ VICTOIRE ! &a+5 Diamants"
  loser:
    commands: []
    message: "&cDéfaite."

Variables : %winner%, %loser%, %arena%

Config — Items des shops

items-shop:
  title: "&d&lItems Shop"
  size: 45
  items:
    mon-item:
      slot: 10
      type: normal     # sword|armor-set|armor-piece|normal
      material: GOLDEN_APPLE
      name: "&6Golden Apple"
      amount: 1
      price: 2
      currency: DIAMOND  # IRON_INGOT ou DIAMOND
      lore: ["&7Description"]

Pour armor-set : utilise chest et legs au lieu de material.

Config — Détection PNJ

shop-detection:
  items-keywords: ["Items", "Combat", "Armes"]
  blocks-keywords: ["Blocks", "Blocs", "Build"]

Architecture

cw_templates/desert/     ← immuable
    → Copie fichiers → cw_game_<uuid>/  ← temporaire
    → WorldCreator → Partie → unload → supprimé

Gamerules auto : pas de mobs, pas de cycle jour/nuit, pas de météo, midi fixe, pas de feu, pas d'advancements.

FAQ

Combien de parties simultanées ?

10-20 avec 4 Go RAM. Limité par RAM et disque.

La TNT casse la map ?

Non. Seuls les blocs joueurs sont détruits.

Crash pendant une partie ?

Dossiers cw_game_* restent. Supprimez-les manuellement.

2v2 / 4v4 ?

Version actuelle = 1v1 uniquement.

Shop vide ?

Supprimez config.yml et redémarrez.

Joueurs tombent dans le vide ?

/cw setlobby à la bonne position.

Nouvelle map ?

Build → cw_templates/ → /cw create → /cw tp → /cw set → /cw enable

Mise à jour ?

Remplacez le JAR. arenas.yml préservé. Supprimez config.yml si nouvelles sections.
