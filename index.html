TrajCakeWars

A complete, production-ready Cake Wars 1v1 minigame engine. Built for performance, customization, and seamless integration with any server.

🏟️

Multi-Arena

Unlimited concurrent games. Each match runs in its own temporary cloned world.

⚙️

Fully Configurable

Shop items, rewards, messages, generators — all in one config.yml.

🔌

Plugin Agnostic

Citizens, FancyNpcs, ZNPCsPlus, Vault, LuckPerms, TAB, or a simple renamed villager.

🔒

Private Sessions

Create invite-only matches with a 6-character code. Challenge your friends.

💀

No Death Screen

Custom death system. No vanilla death screen ever. Smooth respawn with countdown.

🧱

Protected Map

TNT only destroys player-placed blocks. Original terrain is untouchable.

Quick Start

1. Prepare the template

# Manual copy or in-game:
/cw savetemplate desert

2. Configure the arena

/cw create desert
/cw tp desert                   # TP into the template
/cw set desert spawn1
/cw set desert spawn2
/cw set desert cake1
/cw set desert cake2
/cw set desert gen1
/cw set desert gen2
/cw set desert diamond           # Repeat for each gen
/cw set desert lobby
/cw setname desert §6§lDésert
/cw settemplate desert desert
/cw back
/cw setlobby                     # At the lobby return point
/cw enable desert
/cw save

Create a Template

A template is a world folder in cw_templates/. Each game clones it. The original stays intact.

Method 1: Manual copy

serveur/cw_templates/desert/   ← full world folder

Method 2: In-game

/cw savetemplate desert
Important: The template must include shop NPCs (entities are cloned with the terrain).

Configure an Arena

/cw tp <arène> to TP into the template. /cw back to return.

Settings

/cw setname <arène> <nom>          # Display name (supports §)
/cw seticon <arène> <MATERIAL>     # GUI icon
/cw settemplate <arène> <nom>      # Link to template
/cw set <arène> diamond clear     # Reset diamond gens

Game Flow

/cw → Menu GUI → Play / Privée
→ 2 players matched → Clone template → TP
→ Freeze 5s → GO ! → Generators active
→ PvP + Eat enemy cake (7 slices, 1s cd)
→ Cake destroyed → next death = eliminated
→ Winner → Rewards → TP lobby → World deleted

Death System

  • Fatal damage intercepted and cancelled (no death screen)
  • Player set to Spectator with title countdown
  • Respawn with starter sword after countdown
  • If cake destroyed → permanent elimination
  • Anti-double guard: ignored if already spectator

TNT >TNT & Explosions

Only player-placed blocks are destroyed. Template terrain is protected.

Config — Général

settings:
  prefix: "&d&lCakeWars &r&7» &r"
  lobby-world: "world"
  lobby-location:     # Set by /cw setlobby
    world/x/y/z/yaw/pitch

game:
  countdown: 5         # Pre-game freeze
  respawn-time: 5      # Respawn countdown
  max-duration: 600    # Max seconds (0=infinite)

Config — Générateurs

generators:
  iron:
    interval: 40   # Ticks (20=1s)
    max: 32
  diamond:
    interval: 600
    max: 4

Config — Rewards

rewards:
  enabled: true
  winner:
    commands:
      - "give %winner% diamond 5"
      - "eco give %winner% 100"
    message: "&6⭐ VICTOIRE ! &a+5 Diamonds"
  loser:
    commands: []
    message: "&cDéfaite."

Variables : %winner%, %loser%, %arena%

Config — Items des shops

items-shop:

  size: 45

Architecture

cw_templates/desert/     ← immutable
    → Copie fichiers → cw_game_<uuid>/  ← temporary
    → WorldCreator → Game → unload → deleted

Auto gamerules: no mobs, no daylight cycle, no weather, fixed noon, no fire, no advancements.

FAQ

How many concurrent games ?

10-20 with 4 GB RAM. Limited by RAM and disk.

Does TNT damage the map?

No. Only player-placed blocks are destroyed.

Crash during a game?

Temp world folders remain. Delete them manually.

2v2 / 4v4?

Current version is 1v1 only.

Empty shop?

Delete config.yml and restart.

Players fall into void?

/cw setlobby at the correct position.

New map?

Build → cw_templates/ → /cw create → /cw tp → /cw set → /cw enable

How to update?

Replace the JAR and restart. arenas.yml is preserved. Delete config.yml if new sections added.
